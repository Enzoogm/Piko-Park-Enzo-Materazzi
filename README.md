# Piko-Park-Enzo-Materazzi
Che, acá están los comandos que usé así podés iniciar todo en tu compu:
Prendé el servidor:
node server.js

Abrí Google Chrome y entrá a http://localhost:3000

Para iniciar el Joystick (El Gamepad):
Abrí otra terminal nueva y metete en la carpeta de la app:

Levantá el proyecto:
npx expo start
Escaneá el QR con Expo Go en tu celular.

Acordate de buscar tu IP local escribí la IP en la pantalla del celular y ya vas a poder mover el cuadradito en la PC.
